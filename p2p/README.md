# Compartilhamento P2P de MP3 (Python)

Este é um sistema simples de compartilhamento de arquivos usando o modelo Peer-to-Peer (P2P).

Cada programa funciona como:
- **Servidor** (compartilha arquivos)
- **Cliente** (baixa arquivos)

Você pode usar para trocar arquivos `.mp3` entre computadores (ou entre terminais na mesma máquina).

## 1. Abra DOIS terminais
Você precisa de pelo menos 2 peers (nós).

## 2. Execute o programa
Em ambos os terminais:

```bash
python peer.py
